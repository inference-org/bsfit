bsfit.nodes.utils.get_bayes_lookup¶

bsfit.nodes.utils.get_bayes_lookup(percept_space: numpy.array, stim_mean: numpy.array, k_llh: float, prior_mode: float, k_prior: float, prior_shape: str, readout: str)[source]¶

Create a bayes lookup matrix based on Girshick’s paper rows: M measurements cols: N stimulus feature means value: log(likelihood) of percept

Usage:
percept, logl_percept = get_bayes_lookup(
    percept_space=1:1:360,
    stim_mean=5:10:355,
    k_llh=5,
    prior_mode=225,
    k_prior=4.77,
    k_card=0,
    prior_tail=0,
    prior_shape='von_mises',
    )
Returns

percepts (np.array): percept likelihood

Return type

(np.array)
